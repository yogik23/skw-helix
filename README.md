## Step RUN

1. Clone repo dan masuk ke folder
    ```
    git clone https://github.com/yogik23/skw-helix.git && cd skw-helix
    ```
2. Install Module
    ```
    npm install
    ```
3. Submit PrivateKey di `private_keys.txt`
    ```
    nano private_keys.txt
    ```
   format private_keys.txt
    ```
    privateKey1
    privateKey2
    privateKey3
    ```
4. Jalankan bot 
    ```
    npm start
    ```
